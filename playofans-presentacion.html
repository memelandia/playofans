Contexto del Negocio

¿Qué problema resuelve PlayOFans?

Para entender el producto, primero hay que entender el ecosistema en el que opera.

🌐

¿Qué es OnlyFans?

OnlyFans es una plataforma de suscripción donde creadores de contenido (principalmente adulto) cobran una cuota mensual a sus seguidores ("fans") a cambio de contenido exclusivo. Hay más de 3 millones de creadoras activas y genera miles de millones de dólares al año. Las modelos necesitan constantemente nuevas formas de mantener el engagement y retener suscriptores.

🏢

¿Qué es una agencia OFM?

Una agencia OFM (OnlyFans Management) gestiona múltiples modelos: les lleva la estrategia de contenido, responde chats con fans, gestiona precios y promociones. Suelen gestionar entre 3 y 30 modelos. Necesitan herramientas centralizadas para gestionar todo desde un solo panel. PlayOFans ofrece un plan Agency pensado para ellas.

🎯

El problema que resolvemos

Las modelos necesitan engagement: que los fans interactúen, gasten más y no se den de baja. Pero no existen herramientas de gamificación para este nicho. PlayOFans llena ese vacío con juegos como la ruleta de premios: la modelo envía un enlace, el fan juega, gana un premio (foto exclusiva, descuento, videollamada...) y así se genera una interacción emocional que fideliza.

Cómo Funciona

El flujo completo en 5 pasos

Desde que la modelo configura su ruleta hasta que monetiza el engagement de su fan.

1

🎨 La modelo crea un código en su panel

Desde playofans.com/[slug]/admin, la modelo genera un código único para cada fan (ej: VICK8472). Puede asignar nombre del fan, número de tiradas (1-10) y premios personalizados.

Panel Admin → Sección Códigos
2

🔗 Envía el enlace al fan por chat

Se genera automáticamente un enlace tipo playofans.com/[slug]/ruleta?c=VICK8472 junto con un código QR y plantillas de mensaje prediseñadas para WhatsApp, Telegram o el chat de OnlyFans.

Enlace + QR + Plantillas automáticas
3

🎰 El fan abre el enlace y gira la ruleta

El fan abre la URL en su móvil, introduce su código, ve una ruleta animada con efectos visuales y sonido. Al pulsar "Girar", la ruleta se anima hacia el premio que el servidor ya eligió (seguridad server-side). El resultado nunca se determina en el navegador del fan.

Seguridad: premio elegido en servidor, no en cliente
4

🎁 El fan gana un premio

Tras la animación, se muestra el premio con un mensaje personalizado (ej: "¡Enhorabuena María! Has ganado un Pack Fotos Hot 📸"). Aparece un panel de "Mis Premios" con el historial. El fan puede seguir girando si le quedan tiradas.

Mensaje customizable con variable {nombre}
5

💰 La modelo monetiza

La modelo ve en tiempo real (polling 30s) quién giró, qué ganó y cuántas tiradas quedan. Los premios son "promesas" que se entregan vía OF (fotos, vídeos, descuentos, videollamadas...), generando interacción directa con el fan, lo que se traduce en más propinas, renovaciones y ventas de PPV.

Dashboard con analíticas en tiempo real

Sistema de Afiliados

Crecimiento orgánico por referidos

Cada modelo puede referir a otras modelos y ganar comisiones recurrentes + descuentos en su propio plan.

📌 Reglas del programa

  • → 10% de comisión recurrente durante 6 meses por cada modelo referida que pague su plan.
  • → Código de referido automático: REF-[SLUG]
  • → Link de referido: playofans.com/registro?ref=REF-SLUG
  • → Descuento en plan propio según referidos activos:
    1 referido = 10%  ·  2 referidos = 15%  ·  3+ referidos = 20%
  • → Las comisiones se calculan automáticamente al marcar un cobro como pagado en superadmin.

Stack Técnico

Tecnologías y arquitectura

Stack simple, serverless, sin frameworks pesados. Despliegue automático en cada push.

🖥️

Frontend

HTML · CSS · JavaScript vanilla. Sin React, sin Vue, sin frameworks. Ficheros HTML multi-tenant con CSS variables para temas.

⚡

Backend

Netlify Functions (Node.js 18). Funciones serverless para toda la lógica de negocio. CORS configurado por dominio.

🗄️

Base de datos

Supabase (PostgreSQL). Auth + Row Level Security + funciones PL/pgSQL. 13 tablas, 13 funciones, triggers y RLS policies.

📧

Email

Resend API. Emails de bienvenida, aviso de expiración y notificaciones. Dominio hola@playofans.com.

🎨

Gráficos

SVG para la ruleta (antes Canvas). Canvas API para compartir resultados como imagen. CSS animations para partículas y efectos.

🚀

CI/CD

GitHub → Netlify. Cada push a main despliega automáticamente. Cron diario a las 9AM para facturación y expiración.

Flujo de datos simplificado

Fan (browser)
  ↓ POST /api/validate-code
  ↓ POST /api/prepare-spin → premio elegido en servidor
  ↓ 🎰 animación de la ruleta...
  ↓ POST /api/confirm-spin → spin verificado, tirada descontada

Modelo (admin)
  ↓ Supabase Auth (login)
  ↓ GET /api/admin-get-codes
  ↓ POST /api/admin-create-code
  ↓ GET /api/admin-get-spins-live (polling 30s)

Superadmin
  ↓ Auth + Secret + 2FA TOTP
  ↓ POST /api/sa-billing-generate
  ↓ POST /api/sa-billing-pay → auto comisiones afiliados
